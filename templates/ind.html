The Pekzep language mainly uses Linzklā (pronounced LINTS-claah), or Lanermic logogram, a logographic writing system used mainly in Faikleone, a conworld that we are building. Since Linzklar is (obviously) not in Unicode, we provide each logogram not only by an image representing how it is written but also with a conventionally determined mapping to a Chinese character that conveys roughly the same meaning. Hence , which means "to cut" or "a knife" is annotated as 刀, meaning "a star" or "a year" is shown as 星, and so on. An Anki deck for each Linzklā character is available here.
牌语主要使用磷字（发音为“磷兹克拉”或“磷饅”），这是一种主要用于怀克勒窝内（Faikleone）的语素文字。怀克勒窝内是我们正在构建的一个想象世界。由于磷字不在统一码中，我们不仅为每个文字提供了代表其书写方式的图像，而且还提供了一个按照惯例确定的与汉字的映射，以表达大致相同的意思。因此，意味是“切割”或“刀”的被注释为“刀”，意思是“一颗星”或“一年”的被显示为“星”，以此类推。记忆各磷字的 Anki 卡组在这里!
パイグ語は主に燐字という、私たちが創作中の架空世界ファイクレオネで主に使われている表意文字体系を使用しています。燐字は当然ながらUnicodeにないため、それぞれの燐字には、その字形を表す画像だけでなく、たとえば「切る」「刃物」という意味のは「刀」、「星」や「年」という意味のには「星」、といった具合に、似た意味を持つ漢字で燐字を表した漢字転写も用意しています。燐字とそのパイグ語での発音を覚えるためのAnkiデッキはこちらからダウンロードできます。燐字については、「玄人のための燐字概要」などもご参照ください。
