我須為在乎？

pai2 ly zau aim2 yn2?

Is there anything I should do? / 有什么事是我该做的吗？

彼須為在多。

zap2 ly zau aim2 taun1.

She has a lot to do. / 她有很多事要做。

男須労。

ki1 ly naip2.

A man should work. / 男人应该工作。

此時彼須在来終。

ka1 kak zap2 ly aim2 sak2 ta.

By now, he should have already arrived. / 现在他应该已经到了。

周此時彼須来。

cei2 ka1 kak zap2 ly sak2.

He should arrive around now. / 他这个时候该到了。

我須使言機言彼。

pai2 ly sui1 zep1 kik1 zep1 zap2.

汝須心汝力真。

mua2 ly hia1 mua2 py put.

汝須学多。

mua2 ly nui2 taun1.

You should study hard. / 你应该努力学习。

我等須寝。

pai2 ge ly hok1.

我須加寝。

pai2 ly lin mok1 ka1 pia2.
