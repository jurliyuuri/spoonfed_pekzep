kua2筆verbto write

我無識筆牌言之術。

pai2 mun1 can2 kua2 pek2 zep1 a sit1.

I do not know how to write Pekzep. / 我不会写牌语。

筆哩言無硬。

kua2 lip zep1 mun1 pek1.

Writing Linepāine is not difficult. / 写哩语不难。

彼筆彼名此硬。

zap2 kua2 zap2 sin1 ka1 pek1.

He can hardly write his name. / 他很难写他的名字。

彼筆極美之字。

zap2 kua2 kit hem1 a man1.

He writes characters very clearly. / 他的字写得很清楚。

筆清字此極硬。

kua2 lin man1 ka1 kit pek1.

Writing Linzklā characters is very difficult. / 写磷字很难。

彼力筆清字。

zap2 py kua2 lin man1.

He knows how to write Linzklā characters. / 他会写磷字。

「筆清字此硬乎？」「在。我無識清字。」

"kua2 lin man1 ka1 pek1 yn2?" "aim2. pai2 mun1 can2 lin man1."

"Is it difficult to write Linzklā characters?" "Very difficult. I don't know Linzklā characters." / 「写磷字难吗?」「很难。我不认识磷字。」

我之男上輩識筆牌言之術。

pai2 a ki1 sau2 huok1 can2 kua2 pek2 zep1 a sit1.

My older brother knows how to write Pekzep. / 我哥哥会写牌语。

彼在筆字集。

zap2 aim2 kua2 man1 dat2.

彼在筆一絵。

zap2 aim2 kua2 et2 lek1.

hia1 kua2 ut2 sin1 a.

目汝筆之論書別而無善識来。

ta1 mua2 kua2 a zui1 ak1 pau1 ua mun1 kait can2 sak2.

tut1 kia1 zap2 kua2 a ak1 guk2 ta1.

The book that he wrote is read every day. / 他所写的书每天都被人阅读。

zap2 sui1 io1 kua2 lek1.
